# fleeting_notes_flutter
Fleeting Notes - is a reference for the ideas in your mind

## Running Application for dev
```
flutter run
```
Or as a web-server
```
flutter run -d web-server
```

## Testing Applications on browser
```
flutter test --platform chrome
```
